## Description
There is test task similar to employee management app.

## Task's functional
- Add emloyee
- Delete employee
- Edit employee's data:
  - Fullname
  - Position
  - Birthday
  - Gender
  - Status (working or not)

## Stack Techology
The test task was created with using the following technology stack:
- React
- Redux
- React-Redux
- React-dom
- Moment
